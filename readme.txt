Projeto: Sorteador
Criado por: Wellington da Silva Lima

--------------------------------------------------
Descrição:
--------------------------------------------------
O Sorteador é uma aplicação simples que permite realizar sorteios de 
números de forma aleatória. Desenvolvido com HTML, CSS e 
JavaScript, este projeto tem o intuito de demonstrar conceitos 
básicos de manipulação de eventos, DOM e responsividade.

--------------------------------------------------
Funcionalidades:
--------------------------------------------------
- Geração aleatória de números.
- Interface limpa e responsiva para fácil uso.
- Código organizado para facilitar a manutenção e possíveis 
  extensões do projeto.

--------------------------------------------------
Tecnologias Utilizadas:
--------------------------------------------------
- HTML: Estrutura da aplicação.
- CSS: Estilização e definição de responsividade.
- JavaScript: Lógica de geração aleatória e interação com o usuário.
